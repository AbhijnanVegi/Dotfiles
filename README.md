# Dotfiles

## Requirements

- i3-gaps

- maim
- xclip
- polybar
- rofi
- Hack Nerd Fonts
- picom-ibhagwan-git
- Betterlockscreen
- nm-applet
- kitty
- feh
- zathura (pdf-viewer)
- qt5ct & kvantum for qt theming
- lxappearance for gtk theming
- [GTK Theme](https://github.com/EliverLara/Sweet)

## Instructions

1. Copy i3, picom, polybar, rofi to ~/.config
2. Copy Scripts, Pictures to ~
3. Install all dependencies
